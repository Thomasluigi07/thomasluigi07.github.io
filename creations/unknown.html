SUPER THOMAS 64 2 (working title) (64DD)

Thomasluigi07

The long awaited and very ambitious sequel (nobody has been waiting for this) to SUPER THOMAS 64... is coming eventually. Introducing SUPER THOMAS 64 2 (working title)!!! The shitty self-insert romhack is BACK, but this time with VENGANCE... after the original SUPER THOMAS 64 was cancelled back in 2020.

This romhack is PAL ONLY and (when I figure it out) it will require the 64DD at a certain point in the game. Why? Well you'll see when you eventually play the romhack. New game mechanics include: No drowning, Poison gas meter, UI plagarised from Klonoa 2, 3 health system and so much more...

When will this romhack release? Uhhh... someday. This romhack is FULLY console compatible and is recommended to be played on a PAL system due to NTSC not being properly tested. Using a real 64DD is not recommended as there may be no way to create a custom diskette unless you have a DEV system, so use a flashcart such as a SummerCart64 instead. If you don't have a N64, you must use an emulator that supports combo N64 and 64DD games, such as Project64, mupen64plus and Ares.

What's the current progress you might (not) be wondering? Sound effects are pretty much done and gameplay is also somewhat set in stone. The main focus right now is getting the 64DD to communicate to the console other than just it saying if it's connected or not. Once that is done I plan on fully beginning proper development.

I plan on releasing the romhack on my website, GitHub with full source code available, the Internet Archive (including all builds made before release), and on Romhacking.com (if they even allow it due to the 64DD requirement later on in the romhack)
